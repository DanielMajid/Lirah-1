# Lyre-1 (NTS-1 mkII osc)

Generated by nts1_mkii_port_agent.sh.

## Build

make
make install
make package

## What this scaffold provides

- mkII-compatible Makefile wrapper using the official logue-sdk template flow.
- Unit header metadata in src/header.c.
- Compatibility bridge in src/unit.cc that maps legacy NTS-1 mkI callbacks to mkII runtime callbacks.

## Important limitations

- Binary-only conversion is not possible: .unit and .nts1mkiiunit files must be rebuilt from source.
- The generated bridge is an 80% starting point. You still need to validate DSP behavior, parameter scaling, and CPU usage.
- Oscillator ports can require additional tuning due to fixed-point vs floating-point pipeline differences.
